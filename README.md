# codex-ai-rd-team

多渠道实现仓库（Monorepo）：

- `python/`：Python 编排内核实现
- `codex/`：Codex 原生工具编排实现
  - `rd-team-native/`：纯 Codex 原生版本（不依赖 python 编排器）

## 当前状态

- Python 通道：可通过 `python/bin/ai-rd` 使用
- Codex 通道：可通过 `codex/rd-team-native/bin/ai-rd-native` 使用

## 项目来源

本项目源于 CodeBuddy 版本的 `ai-rd-team`，参考仓库：
